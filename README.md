goland
======

a rogue-like terminal game in go

## Installation:

```
go get -u github.com/mischief/goland/client
go get -u github.com/mischief/goland/server
```

## Running:
Start the server:
```
$GOPATH/bin/server -config $GOPATH/src/github.com/mischief/goland/server/config.lua
```

Start the client:
```
$GOPATH/bin/client -config $GOPATH/src/github.com/mischief/goland/client/config.lua
```

## Debugging:
Tail the logfiles.
